FAQ
Preguntas frecuentes
Funciona para academias online, presenciales o mixtas?⌄
Si. El agente conoce la modalidad de cada curso y adapta la informacion. Para online incluye los detalles de acceso a la plataforma. Para presencial la direccion y requisitos de material.
El agente puede gestionar las becas y la financiacion?⌄
Si. Configuramos todas las opciones disponibles: FUNDAE para empresas, becas por desempleo, pago fraccionado. Cuando el lead menciona el precio, el agente presenta las opciones automaticamente.
Como funciona el recordatorio para clases online?⌄
24h antes el alumno recibe el recordatorio con el enlace de acceso a la videoconferencia o plataforma. 2h antes, otro recordatorio con el mismo enlace. Esto reduce los no-shows online del 30% al 5%.
Funciona para academias con varios cursos simultaneos?⌄
Si. Cada alumno tiene su propio hilo en el historial y recibe recordatorios solo de sus clases. El sistema gestiona tantos cursos y alumnos como necesites.
El seguimiento dias 1/3/7 se puede personalizar por tipo de curso?⌄
Si. El contenido de cada mensaje lo genera GPT con el nombre del alumno y el curso especifico en que esta interesado. No es un mensaje generico, es un mensaje adaptado a cada lead.
Hay permanencia?⌄
No. Podeis cancelar con 30 dias de preaviso. El pago de configuracion no es reembolsable.
